Especificaciones

Construcción y DSP de la Cabina

El software cuenta con un compilador integrado para Windows que permite empaquetar todo el sistema de audio, librerías complejas y DLLs en un ejecutable portátil autónomo.

Ecualizador Rack Paramétrico activo, filtros Butterworth de orden 2, corte variable
Latencia de Mezcla Ultra-baja latencia en búfer circular (< 20ms de retardo en DSP)
Compresión de Voz Puerta de ruido integrada (-60dB a 0dB) y normalizador RMS
Bases de Datos SQLite3 relacional estructurada con índices binarios rápidos
